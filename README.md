# PRODIGY_WD_04 — Personal Portfolio Website

## Task
Build a personal portfolio website showcasing skills, projects, and accomplishments as a web developer, with a home page (headline, photo, skills summary), an About Me section, and project showcase.

## Tech Used
- HTML5
- CSS3 (Flexbox, sticky nav, responsive grid)
- Vanilla JavaScript (active section highlighting)

## Features
- Sticky navbar with smooth-scroll links
- Hero section with profile photo, headline, and CTA
- About Me section with education/experience summary
- Skills grid
- Project showcase linking out to GitHub repos (Tasks 1–3)
- Contact section with email, LinkedIn, and GitHub links

## ⚠️ Before Deploying — Personalize These
- Replace `Your Name` throughout `index.html`
- Replace the placeholder profile image URL with your own photo
- Fill in your real education/location/experience in the About section
- Update the GitHub/LinkedIn/email links with your real profile URLs

## How to Run
1. Clone this repo
2. Open `index.html` in any browser
